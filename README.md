# carlton  
The successor to LadiBE, the worst web server ever.  
Carlton is a basic web server that only supports `HTTP 1.1`.  
# features  
- [x] Supports html files and text  
- [ ] Supports images  
- [ ] Supports videos   
- [ ] Multithreaded (soon)  
# compiling  
Use the `compile` file provided or use clang or gcc to compile it.  
`gcc main.c -o main`  
`clang main.c -o main`
